## 📋 專案簡介

日本一番賞（扭蛋/盲盒）商品監控系統。追蹤最新商品、官網資訊、主圖展開。

**核心功能**：
- 🎁 10 個品牌監控
- 📸 主圖自動抓取
- 📅 發售日期提醒
- 🔔 新品通知

**監控品牌**：日本主要一番賞發行商
